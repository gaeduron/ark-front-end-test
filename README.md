To start the app locally:

1. Clone the repo
2. Run `npm install`
3. create a `.env` file at the root of the ark-front directory with the following value
```
ALCHEMY_API_KEY="Your_Api_key"
```
4. Run `npm run dev`
5. Access the app on http://localhost:3000
